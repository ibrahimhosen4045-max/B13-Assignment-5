- 1️⃣ What is the difference between var, let, and const?
Ans/. var is function-scoped, while let and const are block-scoped.

- 2️⃣ What is the spread operator (...)?
Ans/. The spread operator is used to expand iterable into its individual elements.

- 3️⃣ What is the difference between map(), filter(), and forEach()?
Ans/. map() transforms element, filter() selected element, and forEach() iterates without returning a new array.

- 4️⃣ What is an arrow function?
Ans/. An arrow function is a concise syntax for writing function expressions.

- 5️⃣ What are template literals?
